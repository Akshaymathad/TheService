# The Service - Landing Page

A high-performance, responsive landing page for [The Service](https://theservice.co.in/).

## Features

- **Zero Build Step**: Uses Tailwind CSS via CDN for instant deployment.
- **Mobile First**: Fully responsive design with a custom mobile menu.
- **Performance Optimized**: Minimal external dependencies, semantic HTML, and optimized assets.
- **SEO Ready**: Includes Open Graph, Twitter Cards, and JSON-LD structured data.

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/the-service-landing.git
   cd the-service-landing
   ```

2. Open `index.html` in your browser to view the site.

## Deployment

See [DEPLOY.md](DEPLOY.md) for detailed deployment instructions for GitHub Pages, Vercel, Netlify, and Cloudflare Pages.

## Structure

- `index.html`: Main landing page.
- `js/main.js`: Interaction logic (mobile menu, animations).
- `api/ping.js`: Example serverless function for Vercel/Netlify.
